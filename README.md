## 为什么是播客？

> Sam Altman 不会写一篇 5000 字的博客告诉你 OpenAI 的真实战略，但他会在 Dwarkesh 的麦克风前聊 3 小时；月之暗面杨植麟不会接受 36kr 的快采访，但会和张小珺录一档 4 小时的深度对话；Karpathy 不会写技术博客解释他为什么从 Tesla 走人，但他在 No Priors 上一句话就讲清楚了。

文字访谈被压缩、被编辑、被改写；而播客是原始素材，你听到的就是他们脑子里真正在想的事。

而且播客天然伴随 AI 时代——通勤、洗碗、跑步、做饭这些时间本来就是被浪费的，听播客等于零成本获取信息。一周 5 小时通勤听完 3 期 No Priors，你对硅谷 AI 投资的判断会比 90% 看科技新闻的人更准。
