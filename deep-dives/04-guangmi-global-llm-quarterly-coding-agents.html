一、嘉宾与栏目背景

广密（化名）是拾象 CEO，也是"全球大模型季报"系列的常驻嘉宾。拾象是一家专注于 AI 领域的投资机构，广密以每季度一次的频率与张小珺对谈，梳理全球大模型产业格局。这个系列已经成为中文 AI 圈最有影响力的产业分析栏目之一。

本期是第9集。与往期相比，本期最大的不同是：广密旗帜鲜明地将 Coding 定位为"AGI 第二幕"的核心驱动力，并对硅谷"御三家"（Anthropic、OpenAI、Google Gemini）以及 Meta、xAI 的竞争态势做出了极为犀利的拆解。

二、核心判断：Coding是AGI的加速器
2.1 为什么 Coding 如此重要

广密的核心比喻：

"Coding 就像 Amazon 最早卖书一样——借助卖书把整个仓储、物流、用户全拉通了。Coding 把 AI 从 Chatbot 推向了能干的 Agent 第二幕。"

展开来说：

  • Coding 是"虚拟子宫"：就像 DeepMind CEO Demis Hassabis 将世界模型描述为孕育 AGI 的"虚拟子宫"，广密认为 Coding 承担了同样的角色。通过 Coding，Agent 可以在数字世界实现人类能在电脑和手机上操作的所有行为。
  • 语言 + 代码 + 工具 = 数字世界的完整行动能力：人和动物最大的区别在于语言和工具使用。当 AI 同时掌握了语言、代码和工具调用能力，它在数字世界的行为空间就和人类一样广阔——这就是 AGI 的操作性定义。
  • Coding 是"又长又密"的训练数据：Code 是唯一能提供超长上下文（128K → 1M tokens）且信号密集的训练数据。书籍虽然也长，但信号太发散。
2.2 研究员已经不亲自写代码了

广密指出一个已经发生但很多人还没意识到的变化：

  • 顶尖 AI 研究员已经不再亲自写代码
  • 他们使用 Coding Agent 来放大自己的生产力
  • 这形成了一个正向飞轮：更好的 AI → 更好的 AI 研究员生产力 → 更快造出更好的 AI
"Better AI makes better AI。这就像马车换汽车——最早的汽车经常抛锚，快马还能追上。但一旦汽车稳定跑起来，就再也追不上了。"
2.3 顶尖1%人才的放大效应

关键数据：顶尖1%人才通过 Coding Agent 赋能，生产力放大 10-50 倍。这意味着 AI 不是平均地提升所有人的生产力，而是对最顶尖的人才产生最大幅度的放大——这本质上是一个"精英加速器"。

三、硅谷御三家：逐家拆解

🏆 Anthropic：节奏全对，All in Coding

广密评价：当下最正确的战略选择

Anthropic 的战略是硅谷最清晰的：All in Coding，放弃多模态、放弃 ToC、不做独立推理模型。

  • 创始人亲自看训练数据：这是 Anthropic 内部文化的一个关键细节——Dario Amodei 本人会亲自审查训练数据，这种深度参与在大公司几乎不可能。
  • Underdog 文化：Anthropic 偏好"逆袭者"叙事，面试极为严格，文化凝聚力极强。Meta 花15亿美元试图挖人都被拒绝。
  • Claude Code 的爆炸增长：仅用三四个月，ARR 就超过了 Cursor。按照当前的增速，年底可能冲击15-20亿美元。这是比 ChatGPT 更快的增长速度。
  • Opus 4.5 → 4.6 是跨代际提升：广密将其类比为 GPT-3 → GPT-4 级别的跨越。并预测在2026年6月之前可能还会再有一次同等级别的跨越。
  • 营收神话：从2023年底 ARR 不到1亿美元，到2024年底 9.5亿美元，再到2025年可能超过120亿美元、2026年冲击300-400亿美元。每月营收和 token 消耗的复合环比增速都有 20-30%。

风险：如果 OpenAI 或 Google 举全公司之力做 Coding，Anthropic 会有压力。但广密判断这两家"想要的太多"。

🔶 OpenAI：吃老本，但品牌壁垒不可撼动

广密评价：过去成功的秘诀是下个时代的毒药
  • "真正意义上的 GPT-5"：OpenAI 即将发布的新模型被广密称为"真正意义上的 GPT-5"，这意味着接下来的发布很重要。
  • 战略失误：专注 ToC，忽视 Coding：OpenAI 过度专注于 ChatGPT 的 C 端产品，在 Coding Agent 上落后于 Anthropic。
  • "过去的成功秘诀是下个时代的毒药"：OpenAI 在 Chat 时代的成功——品牌、用户基数、市场份额——可能反而让它在新范式（Agent 时代）中反应迟缓。
  • 品牌壁垒深于技术壁垒：ChatGPT 现在是"AI 的可口可乐"——即使别人配出一样的饮料，大多数人还是选可口可乐。ChatGPT 的留存数据"好到仅次于微信这种社交产品"。
  • 非共识：Coding 和模型的壁垒远没有 ChatGPT 的品牌壁垒高。Sam Altman 的两个平行愿景——10亿 DAU + AGI——正在变成现实。

优势：广密认为 ChatGPT 身上有很多不可复制的时间窗口下的品牌价值。即使模型落后3-6个月，用户也不会流失。

🔵 Google Gemini：最领先的追随者

广密评价：Coding严重落后是最大战略失误
  • Gemini 3 被高估：广密对 Gemini 3 的评价低于市场预期，认为其实际表现不如宣传。
  • Coding 严重落后：这是 Google 最大的战略失误。在 Agent 时代，如果不能在 Coding 上追上来，位置会很危险。
  • 后劲可能最强：从 TPU 芯片到 Gemini 模型到 Agent 应用到 Google 文档、Chrome 浏览器、安卓、YouTube——Google 的纵向整合能力无人能及。即使模型落后几个月，后劲可能是最强的。
  • 掉队的可能性低：作为"最领先的追随者"，Google 的规模优势和资源决定了它不会轻易被甩开。
四、四号位与五号位：Meta 与 xAI
4.1 Meta：已取代 xAI 成为硅谷4号种子
  • Meta 的"超级智能"团队短期内想进全球前三还很难。
  • 核心问题：到底是智能为先还是产品为先？如果是智能为先，水平还在 Grok 后面；如果是产品为先，Meta 的 0-1 基因很弱。
4.2 xAI（Grok）：短期基本掉队
  • 最大问题：战略摇摆。Musk "什么都想要，什么都没 bet"。
  • 做 Chat 赢不了 ChatGPT，做搜索面临 Google 和 Perplexity，做 Coding 有 Anthropic。
  • Grok 模型花了别人10多倍的资源，但没带来10多倍的好处。广密甚至说"一定程度上豆包比 Grok 好用"。
  • 广密预测 xAI 在未来半年到一年内"很大概率并入 Tesla"。
五、模型正在成为新一代 OS

广密在这个问题上有一个清晰的框架：

"语言即世界，代码即方案。如果领先的模型公司不重视 Coding，大概率会掉出第一梯队。"
5.1 为什么是 OS
  • 真正的 OS 是能让应用无限扩展的平台：今天只有 Windows、iOS、Android 能做到这一点。大模型的 Coding 能力正在让 AI 成为新一代的 OS——通过 Coding API，应用可以无限扩展。
  • ChatGPT 就是下一个 Google：搜索 + 问答 + Agent 的横向全家桶。
  • Anthropic 就是下一代 Windows：通过 Coding API 让应用无限扩展——"之前很多人说自己是 OS，但都不是真 OS"。
5.2 投资配置

广密给出了他的"100块钱配置"：

  • 40块 OpenAI：ChatGPT 的非技术壁垒（品牌、用户、留存）远超 Coding 的技术壁垒。
  • 40块 字节：豆包是中国版 ChatGPT，拥有类似的分发和品牌优势。
  • 10块 Anthropic：Coding Agent 最强，但技术壁垒不如品牌壁垒牢固。
  • 10块 Google：规模效应最强，后劲足。
六、宏观判断：白领通缩窗口已经开启

广密在访谈中提出了一个沉重的宏观判断：

"白领通缩与失业窗口已经开启。"
  • 过去一个季度的智能水平进步幅度赶上了2025年全年
  • 当顶尖1%的人通过 Coding Agent 实现10-50倍生产力提升时，这意味着大量白领工作的边际价值正在急剧下降
  • AI 不是平均地"替代"工作——它是精英加速器，先放大塔尖，再压缩塔基
七、关键数据一览
  • Opus 4.5 → 4.6：GPT-3 → GPT-4 级别的跨代际提升
  • 智能进步速度：一个 Q 赶上 2025 全年
  • Coding Agent 生产力放大：顶尖1%人才提升 10-50 倍
  • Anthropic 营收：2024年底 9.5亿 → 2025年可能超120亿 → 2026年冲刺300-400亿
  • 每月 token 消耗复合环比增速：20-30%
  • OpenAI ARR：120亿美元
  • Claude Code ARR 增长：三四个月反超 Cursor，年底可能冲击15-20亿美元
  • 前两名（OpenAI + Anthropic）占 AI 产品营收：70-80%

🏁 一句话总结

Coding 正在把 AI 从聊天工具变成能干活、能自我迭代的生产力引擎。在这场范式切换中，Anthropic 的战略判断最准（All in Coding），OpenAI 的品牌护城河最深（ChatGPT = AI 的可口可乐），Google 的后劲最足（纵向整合无人能及）。而 Coding 的崛起也意味着：白领工作的大规模重组已经开始了。
