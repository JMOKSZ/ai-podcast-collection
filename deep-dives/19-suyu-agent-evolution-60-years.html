Logical Agent 時代（1960s–1990s）

第一代 Agent 完全基於符號邏輯。你俾佢一個目標，佢用邏輯推理去分解成一系列 action。經典工作包括 STRIPS 規劃器、SOAR 架構、定理證明器等。呢個時代嘅 Agent 極度脆弱——世界必須完美建模，任何 uncertainty 都會令成個規劃崩塌。

蘇煜指出，呢個時期嘅 Agent 雖然難以落地，但留下咗重要嘅概念遺產：Planning as Reasoning、Goal-Oriented Decomposition、Means-Ends Analysis。呢啲概念喺六十年後嘅 Language Agent 中以新嘅形式重生。

Neural Agent 時代（2000s–2010s）

神經網絡崛起後，Agent 研究嘅重心轉向 Deep Reinforcement Learning。AlphaGo、DQN 喺遊戲領域取得突破性進展。但呢啲 Agent 嘅限制好明顯：只能在限定環境中工作，每一次都要從頭訓練，缺乏泛化能力。

蘇煜認為 Neural Agent 嘅核心問題係：樣本效率極低，且難以融入語言。一隻打 Atari 遊戲嘅 DQN 耗費上千萬幀訓練，但換個遊戲一切重來。語言作為人類最自然嘅知識傳遞媒介，好難被 neural agent 有效吸收。

蘇煜提出咗一個極有說服力嘅觀察：OpenClaw Moment 同 ChatGPT Moment 有非常多相似嘅地方。

ChatGPT Moment：自然語言輸入→模型直接輸出文本，中間跳過了結構化編程的步驟。
OpenClaw Moment：自然語言輸入→模型直接操作電腦，中間跳過了編寫 GUI 或 API 調用代碼的步驟。
