開篇：「今夕是何年」——金曲獎與文藝復興嘅隱喻

莊明浩以金曲獎頒獎禮上「老傢伙回歸」嘅畫面開場，引出呢期節目嘅核心隱喻：文藝復興。

「所以什麼是文藝復興？」

文藝復興唔只係藝術嘅復興，而係人類文明一次徹底嘅重估——從宗教到科學，從藝術到政治，所有嘅框架都被打破同重建。莊明浩用呢個框架來理解 2026 年嘅 AI 行業：一切都在被重估，從資本配置到智能定義，從組織形式到人類自身嘅位置。

佢嘅 PPT 封面引用咗文藝復興時期嘅透視法發明（線性透視）作為類比——正如透視法徹底改變咗繪畫，世界模型（World Model）可能徹底改變 AI。

「一切都在重估之中。從資本到智能，從 Agent 到治理，所有框架都在被改寫。」

第一章：資本與硬件——美第奇的賬本

莊明浩將呢章命名為「美第奇的賬本」——美第奇家族用財富贊助文藝復興，今日嘅科技巨頭用 CAPEX 贊助 AI 革命。但賬本嘅背面，係令人不安嘅數字。

AI 嘅敘事已經徹底改變

唔再係「AI 能做什麼」，而係「萬億美金的流轉」。Token 買方同賣方嘅利益博弈成為市場主線。

存儲行業的壟斷結構

莊明浩花咗唔少篇幅講存儲：

  • 存儲行業係壟斷嘅——HBM 技術高度集中
  • SK 海力士去美國上市，呢個決定「意味深長」
  • 存儲行業嘅周期喺美光毛利率上清晰可見
  • 「提及存儲要先看成本佔比和絕對量增長」
📊 關鍵數字：Intel 同思科嘅股價「漫漫回歸路」係經典敘事坍塌案例。十年前嘅王者今日嘅處境，值得每一個 AI 投資人細味。

賬本的背面——債務問題浮現

莊明浩反覆使用幾個詞：

  • 「收入只夠折舊」——資本密集行業嘅典型困局
  • 「現金流出了問題」——高速增長掩蓋嘅結構性問題
  • 「債」頻繁出現——neocloud 嘅挑戰被債務放大
「第一章的結論到底是黃金時代還是泡沫破裂之前？」

莊明浩冇俾出確定答案，但佢埋落嘅問題種子好有力：呢個係基礎設施嘅黃金時代，定係泡沫破裂嘅前夜？

第二章：智能的文藝復興

美國頭部模型廠商嘅競爭格局

  • 模型能力「已經非常強」
  • 發布節奏「瘋狂加速」——模型競賽進入白熱化
  • 多模型共存成為常態——唔止係技術判斷，更加係投資判斷
  • Neolabs 興起：大量新實驗室從大廠出走成立，形成新嘅創新動力
  • Anthropic 超過 OpenAI——2026 年最大嘅格局變化之一
「瘋狂的數字：1 萬億美金」——對 AI 產業總市場規模嘅估算

中美對抗

  • 中國模型廠商喺錢嘅角度有差距，但效率路線形成咗差異化
  • 中美 AI 生態正形成兩條截然不同嘅技術路線：閉源前沿模型 vs 開放權重、高效率擴散
  • 開源同閉源唔係非此即彼——兩種路線正在分化

世界模型：AI 嘅「透視法」

莊明浩引用李飛飛對世界模型嘅分類（A Functional Taxonomy of World Models）：

  • 渲染器（Renderer）：給眼睛看嘅生成畫面
  • 計劃器（Planner）：符合物理規律嘅推演
  • 模擬器（Simulator）：理解完整世界常識

呢個分類框架已經成為行業共識。莊明浩認為世界模型可能係推動 AI 從「預測下一個詞」到「理解同模擬現實世界」嘅關鍵。

第三章：Agent 元年——畫匠與工坊

呢一章係全期信息密度最高嘅部分：

Codex = 新嘅 ChatGPT

「Claude Code 時刻帶來了范式轉變。」

Codex 嘅爆發唔只係一個產品嘅成功，而係「用戶層面的時代變了」——從 GUI 到 CLI，從被動應用到主動 Agent，呢個轉變嘅速度遠超預期。

Code 吞噬一切

  • Coding 成為核心戰場
  • 軟件生態正在被改寫
  • Agent Infra 蓬勃發展——從模型到產品，從產品到服務
  • 模型廠商嘅邊界不斷擴展
「Codex 可能定義了這一輪的產品樣式。」

國內嘅共識形成

  • 國內大廠再次掀起競爭
  • Codex 定義咗呢一輪嘅產品樣式
  • 變化同調整也在發生
  • 市場規模變化——但「模型廠商吃一切」嘅擔憂也在加深
🔑 關鍵判斷：莊明浩認為 2026 年可能係 Agent 從「聊天」走向「任務執行」嘅關鍵年份。唔再係 demo 同概念，而係真實嘅工作負載。

第四章：治理與反彈

大模型也要「先審後發」嗎？

莊明浩提出三對博弈：

  1. 安全 vs 創新速度
  2. 國家主權 vs 全球協作
  3. 技術樂觀主義 vs 技術懷疑論

Tokenmaxxing 嘅敘事「很快過去了」

因為 Token 仲未可以代表真實價值。當市場注意力從「堆 Token」轉向「ROI」，AI 產品嘅真正價值衡量標準變得模糊。

第四支柱嘅缺失

莊明浩認為：技術、資本、知識之後，AI 時代需要嘅「第四支柱」——超越呢三者嘅某種人類特質——仍然缺失。

「人還是萬物的尺度嗎？——這個問題不再只是哲學問題。」

2050 年嘅一種可能

莊明浩喺結尾拋出一個令人深思嘅畫面：「當智能不再稀缺，人類的位置在哪裡？」

呢個問題唔係修辭——係成個行業面對嘅終極命題。如果智能變成免費、充足、超越人類嘅時候，人類嘅價值同位置應該點樣重新定義？

「2050 年的一種可能——當智能不再稀缺，人類的位置在哪裡？」

五大隱藏主題線

除咗四章嘅表面結構，莊明浩嘅 PPT 仲貫穿着五條隱藏主題線：

1. 「剛剛開始」

反覆出現嘅語氣——AI 產業無論係資本泡沫、模型競賽定係應用落地，全部都只係「剛剛開始」。序幕階段，唔係落幕。

2. 文藝復興作為認知框架

唔只係一個好聽嘅名字。莊明浩用文藝復興嘅結構來理解 AI：贊助人（資本）、復興（智能）、工坊（Agent）、治理（教會同國家嘅新博弈）。呢個框架比單純嘅「技術演進」敘事更加豐富。

3. 資本結構嘅脆弱性

從 neocloud 嘅債務到港股嘅解禁潮，從三位數 PS 倍數到「收入只夠折舊」，莊明浩對資本市場嘅擔心係實質嘅——唔係「AI 泡沫會否破裂」，而係「破裂嘅時候邊個會死得最慘」。

4. 人類嘅位置被重新定義

由「第四支柱」到「人還是萬物的尺度嗎」，呢條線貫穿全套 PPT。莊明浩認為呢個問題嘅答案會決定 AI 時代嘅政治同社會走向。

5. 中國嘅特殊路徑

效率路線、免費習慣、大廠主導、開源生態——中國 AI 產業正行緊一條同矽谷完全不同嘅路。莊明浩冇做價值判斷，但佢認為呢種分歧會持續加深。

結論：剛剛開始

呢期播客最後一句話係——

「剛剛開始。」

兩個字，但係全套 76 頁 PPT 嘅總結。無論你點樣理解 2026 年上半年嘅 AI 行業——泡沫、復興、混亂、重估——所有嘢都只係序幕。

莊明浩嘅態度比上次（2.5 個月前）更加審慎。上次佢更多關注技術曲線，呢次重點放在咗資本結構同治理問題上。佢反覆提及「剛剛開始」——暗示呢一切只係序幕，真正嘅大戲仲未上演。
